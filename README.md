# chenleishen.github.io

Personal website for Shelley Shen, published with GitHub Pages at:

https://chenleishen.github.io

## Local Development

Run a static server from the repository root:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Structure

- `index.html` - homepage markup
- `style.css` - site styling
- `Image/` - profile photo and social icons
- `resume/` - downloadable resume PDF
